# Correction

## Tableau de notation


| **Critère**           | **Poids** | **Description**                                                                              | **Échelle de notation**                                                                                                                                                                                                                                                                                 | **Note** | **Commentaire** |
| --------------------- | --------- | -------------------------------------------------------------------------------------------- | ------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------- | -------- | --------------- |
| **Stateless widgets** | 2 pts     | Évalue l'utilisation et la gestion des widgets Stateless dans l'application.                 | - Bon (2) : Découpage efficace en sous-widgets<br>- Faible (1) : Utilisation sous-optimale. Nécessite un redécoupage                                                                                                                                                                                    |          |                 |
| **Stateful widgets**  | 4 pts     | Évalue l'utilisation et la gestion des widgets Stateful dans l'application.                  | - Excellent (4) : Utilisation optimale et gestion efficace.<br>- Bon (3) : Bien utilisé mais quelques améliorations possibles.<br>- Moyen (2) : Utilisation acceptable mais quelques bugs d'intéractions.<br>- Faible (1) : Intéractions non fonctionnelles.                                            |          |                 |
| **State Management**  | 10 pts    | Évalue la gestion de l'état dans l'application pour assurer la cohérence et la réactivité.   | - Excellent (8-10) : Gestion d'état optimale et cohérente.<br>- Bon (6-7) : Bien géré mais quelques améliorations possibles.<br>- Moyen (4-5) : Gestion acceptable mais nécessite des améliorations.<br>- Faible (2-3) : Gestion sous-optimale.<br>- Très faible (1) : Mauvaise gestion ou incohérente. |          |                 |
| **API call**          | 2 pts     | Évalue la qualité et l'efficacité des appels API pour la récupération et l'envoi de données. | - Bon (2) : Appels API efficace.<br>- Faible (1) : Appel non fonctionnel ou inefficace.                                                                                                                                                                                                                 |          |                 |
| **Firebase**          | 8 pts     | Évalue la qualité du stockage des données utilisateurs et de leurs récupérations.            | - Excellent (7-8) : Gestion optimale et cohérente.<br>- Bon (5-6) : Bien géré mais quelques améliorations possibles.<br>- Moyen (3-4) : Gestion acceptable mais nécessite des améliorations.<br>- Faible (2) : Gestion sous-optimale.<br>- Très faible (1) : Mauvaise gestion ou incohérente.           |          |                 |
